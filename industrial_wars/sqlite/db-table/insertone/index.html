InsertOne

:InsertOne(data)

Описание#

Вставляет в таблицу данные. Чтобы вставить несколько за раз, можно использовать Insert

Внимание

Для изменения данных, используй функцию Update

Агрументы#

НазваниеТипОписание
datatableТаблица данных которые надо вставить в таблицу. Смотри в Примеры

Возращает#

  1. nil / false - Если false, значит есть ошибка.

Примеры#

Вставляем данные игроков

t:InsertOne({
id64 = "player1",
money = 30000,
lvl = 13,
})
