PrikolMen's Markers

Типы маркера#

НомерНазваниеОписание
0wayПуть
1friendДруг
2enemyВраг
3defenceЗащищать
4attackАтаковать
5itemПредмет

Функции ( Server )#

DropMarker#

Описание#

  • Создаёт маркер там куда смотрит игрок
P_Markers.DropMarker(ply)

Аргументы#

НазваниеТипОписание
plyplayerИгрок.

Send#

Описание#

  • Создаёт маркер
P_Markers.Send(pl, type, pos, ent, ply)

Аргументы#

НазваниеТипОписание
plplayerИгрок который будет видеть маркер.
typestringТип маркера, все типы
posvectorПозиция маркера
plyentityСоздатель маркера

Примеры#

Создаём маркер пути на нулевом векторе

P_Markers.Send(ply, "way", Vector(0, 0, 0), nil, ply)
