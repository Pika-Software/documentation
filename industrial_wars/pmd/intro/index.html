PF - PrikolMen's Framework

Описание#

  • Фреймворк необходимый для всех p_ аддонов, он расширяет базовый функционал GLua и исправляет баги игры.

  • Фреймворк не расчитан на работу вне "Индустриальных войн", пока я не сделаю полную обособленность от ИВ использование фреймворка в других проектах невозможно!

Доступные API#

  • PF - Все существующие на момент написания документации функции и хуки доступные для использования в фрейморке.
  • PMD - Все достуные функции и хуки для "Загрузчика моделей игрока"
  • PPM - Все доступные на данный момент функции и хуки для "Менеджера Игрока"
  • PM - Все доступные функции для "Маркеров"
